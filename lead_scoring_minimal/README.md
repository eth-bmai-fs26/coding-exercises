# Lead Scoring Minimal MVP

A minimal end-to-end lead scoring system using PyTorch for tabular classification.

## Overview

This project demonstrates a complete ML pipeline:
1. Generate synthetic lead data
2. Train a PyTorch neural network to predict conversion probability
3. Run a web UI to score new leads from uploaded CSVs

**Note:** All data is fictitious and for demonstration purposes only.

## Setup

### 1. Create Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

## Usage

### Step 1: Generate Synthetic Data

```bash
python generate_data.py
```

This creates:
- `data/leads_train.csv` - Training data with conversion labels
- `data/leads_to_score.csv` - Unlabeled data for scoring

### Step 2: Train the Model

```bash
python train.py
```

This trains a PyTorch MLP and saves:
- `artifacts/model.pt` - Trained model weights
- `artifacts/preprocessor.joblib` - Fitted preprocessor
- `artifacts/meta.json` - Metadata (columns, dimensions, AUC)

Training includes validation AUC tracking and early stopping.

### Step 3: Run the Web App

```bash
streamlit run app.py
```

The app will:
- Load trained artifacts
- Display expected columns and model performance
- Allow CSV upload for scoring
- Show top 25 leads by score
- Provide download for full scored results

## Expected Columns

### Numerical Features
- `company_size` - Number of employees
- `pages_viewed_7d` - Pages viewed in last 7 days
- `email_opens_30d` - Email opens in last 30 days
- `website_visits_30d` - Website visits in last 30 days
- `days_since_signup` - Days since account creation
- `avg_session_duration_min` - Average session duration in minutes
- `feature_usage_score` - Product feature usage score (0-100)

### Output
- `lead_score` - Conversion probability (0-1)
- `lead_tier` - Risk tier: low (<0.33), medium (0.33-0.66), high (>0.66)

## Requirements

- Python 3.10+
- No internet connection required
- Runs on CPU (no GPU needed)
