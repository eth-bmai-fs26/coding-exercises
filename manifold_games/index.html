Explainer 7
The Autoencoder Problem
φ⁻¹: M → ℝd  ·  not a homeomorphism

Vanilla autoencoders optimize reconstruction loss — but that says nothing about how the latent space is organized. The encoder can fragment a smooth manifold into disconnected clusters, invert local neighborhoods, and create vast empty gaps. Reconstruction succeeds; navigation fails.

fragmentation inversion scale mismatch gaps in latent space VAE motivation

Explainer 8
The Idea of a VAE
φ⁻¹: x → 𝒩(μ, σ²)  ·  encode to a region, not a point

Vanilla autoencoders fragment the latent space. VAEs fix this by mapping each input to a Gaussian distribution — a 1D interval or 2D σ-ball — so neighboring inputs overlap in latent space, enabling smooth interpolation and generative sampling.
