Autoencoder: House Manifold

Key Idea
20D Feature Space, 2D Manifold

Each house is described by 20 features, representing dimensions and colors of its block, window, door, and roof. Despite living in ℝ²⁰, all valid houses lie on a 2D surface M: a single pair (z₁, z₂) determines everything.

Panel A Sample Dataset — Five Houses

Five pre-generated latent points spanning the space. Click any card to jump to that point in the latent explorer below.

Key Idea
φ = Encoder  ·  φ⁻¹ = Decoder

The decoder φ⁻¹: ℝ² → ℝ²⁰ generates all 20 features from latent coordinates. The encoder φ: ℝ²⁰ → ℝ² compresses any feature vector back to the manifold, finding z* = argminz ‖φ⁻¹(z) − f‖² via the pseudoinverse.
