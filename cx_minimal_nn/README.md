# Minimal Neural Network — Iris Classification

A self-contained Colab notebook that trains a small neural network on the classic Iris dataset.

**Notebook:** [notebooks/iris_nn.ipynb](notebooks/iris_nn.ipynb)

## What's covered

- Loading and splitting a dataset
- Defining a 2-layer network with PyTorch (`nn.Module`)
- The training loop: forward pass → loss → backward pass → weight update
- Plotting the learning curve
- Evaluating accuracy on a held-out test set

## Prerequisites

- Python basics (functions, loops, variables)
- No prior ML or PyTorch knowledge required

## Dependencies

```
torch
scikit-learn
matplotlib
```

All are pre-installed in Google Colab.
